## 二、开发阶段

### 阶段一：核心体验完善（v1.5.0 — 当前）

**目标**：完善现有功能的用户体验和交互细节

| 优先级 | 任务             | 说明                                |
| ------ | ---------------- | ----------------------------------- |
| P0     | 项目管理交互优化 | 项目卡片、详情页、创建/编辑表单完善 |
| P0     | 项目评论集成     | 项目详情页集成评论组件              |
| P0     | 里程碑 UI        | 里程碑时间线、进度可视化            |
| P1     | 团队管理前端     | 团队列表、成员管理、邀请流程        |
| P1     | 虚拟滚动优化     | 长列表性能优化                      |
| P1     | 图片懒加载       | 图片按需加载，优化首屏速度          |
| P2     | 富文本编辑器     | 文章创建/编辑集成 Markdown 编辑器   |
| P2     | 深色模式         | 支持浅色/深色主题切换               |

### 阶段二：文章体系重构（v1.6.0）

**目标**：将文章模块扩展为操作手册和通知系统

**背景**：参见后端 [文章体系重构设计方案](../TeamGalleryGo/docs/DESIGN_ARTICLE_REFACTOR.md)

| 优先级 | 任务         | 说明                                         |
| ------ | ------------ | -------------------------------------------- |
| P0     | 首页重定向   | `/` 重定向到 `/projects`                     |
| P0     | 导航栏重构   | 移除文章链接，增加手册链接和铃铛通知图标     |
| P0     | 铃铛+红点    | Bell 图标 + el-badge 未读数徽标              |
| P0     | 操作手册列表 | ManualListView（/manuals）                   |
| P0     | 操作手册详情 | ManualDetailView（/manuals/:id）             |
| P0     | 通知列表     | NotificationListView（/notifications）       |
| P0     | 通知详情     | NotificationDetailView（/notifications/:id） |
| P1     | 未读数轮询   | 每 30 秒轮询未读通知数                       |
| P2     | 通知已读标记 | 单条标记已读 + 全部标记已读                  |

### 阶段三：体验优化（v1.7.0+）

| 优先级 | 任务       | 说明                    |
| ------ | ---------- | ----------------------- |
| P1     | 骨架屏     | 文章/项目列表加载骨架屏 |
| P1     | 响应式优化 | 移动端适配              |
| P2     | 无障碍访问 | a11y 支持               |
| P2     | 离线缓存   | PWA 支持                |
| P3     | 多语言     | i18n 国际化             |

---

## 三、任务优先级

```
高优先级: 项目管理完善 → 文章体系重构
中优先级: 团队管理 → 体验优化
低优先级: 响应式 → 多语言
```
