## 版本格式说明

遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/) 规范：

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

| 类型 | 说明 |
|------|------|
| `Added` | 新增功能 |
| `Changed` | 现有功能的变更 |
| `Deprecated` | 即将移除的功能 |
| `Removed` | 已移除的功能 |
| `Fixed` | Bug 修复 |
| `Security` | 安全修复 |
